# Como rodar

1. Inicia o conteiner com docker compose
2. Dentro do conteiner rode um dos scripts de build, `/workspace# ./scripts/build_linux.sh `
3. fora do conteiner, na pasta build, rode `./rayTracingAlgorithms_client > image.ppm` a casa vai aparecer no arquivo .ppm

se nao funcionar olhe o docs/SETUP.md deve estar faltando libs para compilacao etc
